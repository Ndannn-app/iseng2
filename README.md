# Buat Hari Spesial
Proyek ini adalah halaman web interaktif untuk Valentine. Pengguna dapat memilih "Yes" atau "No", dan tergantung pada pilihan mereka, akan muncul gambar lucu atau pesan spesial.

## Fitur:
- Interaksi dengan tombol "Yes" dan "No"
- Efek hati yang jatuh
- Tema Valentine yang lucu dan manis
